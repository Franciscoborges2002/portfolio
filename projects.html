> School Management System Rework

📚 School Project

In this project we were tasked with passing the project from monolithic to a microservices architecture.

We built a total of 7 microservices, including an api gateway to receive and respond to all request from the clients and a saga orchstrator(api composition). We used docker in all project with docker compose to automate the creation of all system and to ease the process to export all project to a VPS.

This project was one of my favorite in all university path, because building the architectures of projects is one of my biggest passions.

This project was made in the course of Arquitetura de Sistemas de Informação Distribuidos.

Tech used:

  • Backend: Java | Spring (Already given)

  • Rework: Java | Spring

  • Includes: Api Gateway | Api Composition

  • Other Tools: Docker/Docker Compose | UML
